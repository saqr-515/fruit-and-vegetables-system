# Blueprint - نظام صقر لإدارة الحسبة (Smart POS Upgrade)

## نظرة عامة
تطوير شاشة البيع لتصبح مرنة بنسبة 100%، مما يسمح بإضافة الزبائن والأصناف الجديدة "على الطاير" دون الحاجة لمغادرة الشاشة.

## الميزات الفنية المضافة
- **Dynamic Entity Creation:** إنشاء تلقائي للحسابات (Accounts) والأصناف (Inventory) عند ترحيل الفاتورة إذا لم تكن موجودة.
- **Searchable Datalists:** استخدام قوائم بحث ذكية للزبائن والمنتجات لتقليل وقت الإدخال.
- **Auto-Price Fetching:** جلب آخر سعر بيع مسجل للصنف تلقائياً عند اختياره.
